The Present

One thing I definitely didn't expect working on this project was how its scope would expand. It started as a simple site meant to serve a downloadable template and offer some explanation for what was inside. At this point, I think the project is definitely reaching the level of complexity and involvement a project in a 400-level class should. I also didn't expect how much of this project would "iceberg" — that is, most of the work I did wasn't on the final site seen here, but in the research, writing, documentation and designing behind the scenes.

Because of this, many of my goals stated at the midterm were not reached; however, I feel I made strong headway into what, hopefully, will become a very high-quality capstone project. Under the Breakdown section of this site is a line-by-line walkthrough of my current HTML/CSS template — the result of truly an ungodly amount of research. I have listed as many of my sources as I could inline, and on the Sources page. Lastly, the first few articles I have written are up in the Basics page.

The Future

I have several goals in mind for next semester. First, of course, is expanding upon the articles I have written to include the topics I proposed in my midterm. Armed now with the research and sources, this can be done without having to do all the grunt work first. Second, from a front-facing site point of view, I have lots of features in mind that I want to try implementing: a javascript or php download customizer for the template, syntax highlighting for all the code snippets, video tutorials to go alongside the writing, I want to improve and tweak the responsive design, as well as clean up the code (which is pretty messy, as you no doubt can tell).

At the end of next semester, I hope to launch this project as a wholly-professional "product", out there in the wild to be used and improved upon. Towards this, I've already open-sourced the project, and I've already actually hosted this site on my Github account, which has a full record of commits showing how I built the site.
